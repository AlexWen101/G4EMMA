35 // Class Description
36 // Process for screened electromagnetic nuclear elastic scattering;
37 // Physics comes from:
38 // Marcus H. Mendenhall and Robert A. Weller,
39 // "Algorithms for the rapid computation of classical cross sections
40 // for screened Coulomb collisions "
41 // Nuclear Instruments and Methods in Physics Research B58 (1991) 11-17
42 // The only input required is a screening function phi(r/a) which is the ratio
43 // of the actual interatomic potential for two atoms with atomic numbers Z1 and Z2,
44 // to the unscreened potential Z1*Z2*e^2/r where e^2 is elm_coupling in Geant4 units
45 // the actual screening tables are computed externally in a python module "screened_scattering.py"
46 // to allow very specific screening functions to be added if desired, without messing
47 // with the insides of this code.
48 //
49 // First version, April 2004, Marcus H. Mendenhall, Vanderbilt University
50 // May 1, 2008 -- Added code to allow process to have zero cross section above max energy, to coordinate with G4MSC. -- mhm
51 //
52 // Class Description - End
